# Persona Templates Starter Guide

A public starter guide for building reusable AI personas with a stable core, an operating manual, optional extensions, and example patterns.

## What this includes

- `index.html` — public-facing single-page guide
- `.gitignore` — excludes local Vercel output

## Core ideas

- Start with `base-soul.md` for identity and personality
- Add `base-agents.md` for operating rules and SOP
- Layer optional extensions only after the core works
- Use examples as references, not as copy-paste production files

## Recommended workflow

1. Copy `base-soul.md` and `base-agents.md`
2. Rename them for your new persona
3. Rewrite role, mission, and communication style
4. Keep the operating flow stable at first
5. Add extensions only when necessary

## Design direction

This guide uses a dark-first, monospace-led visual system inspired by xAI's minimal design language.
